# dotfiles

## 導入手順

```
 curl https://raw.githubusercontent.com/ohashi-yuki/dotfiles/master/setup.sh | sh
```

又は

```
git clone git@github.com:ohashi-yuki/dotfiles.git
cd dotfiles
make setup
```

curlの方がおすすめです。

以下、手動で設定が必要なもの。  

## solarized

iTerm2 > Preference > Prfiles > Colors > solarized Darkを指定。

## Dock

システム環境設定 > Dock > 画面上の位置 > 左
